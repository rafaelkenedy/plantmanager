🚀 Getting started
Clone the project and access the folder

$ git clone https://github.com/birobirobiro/nlw-05-plantmanager.git && cd nlw-05-plantmanager
Follow the steps below

# Install the dependencies
$ yarn install

# Start the project
$ expo start
